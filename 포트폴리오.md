포트폴리오에 추가해야 할 것들 (우선순위별)
1. 프로젝트 케이스 스터디 (가장 임팩트 큼)
단순 "SMS 모니터링 시스템 개발"이 아니라:

배경/문제: 삼성전자에서 슈퍼컴퓨터 상태를 실시간으로 파악할 방법이 없었다
나의 역할: 실시간 차트 대시보드 설계·개발
기술적 결정: 왜 그 기술을 선택했는지
결과: 삼성전자 관제실 대형 화면에 실제 적용
스크린샷/다이어그램: 시각적 증거
2. 정량적 성과 추가
"시스템 개발" → "OO명 사용자 대상 시스템, 처리시간 OO% 단축" 같은 숫자

3. 개발자 성장 스토리
생산직 → 부트캠프 → 개발자 → PL 전환 과정을 스토리텔링으로. 이게 17년차의 가장 큰 차별점

4. About 개선
생년월일, 전화번호 같은 이력서 정보 빼고 → "나는 어떤 개발자인가" 스토리 중심으로

5. 인터랙티브 요소
스크롤 애니메이션, 기술 스택 시각화(경험 연수 바 차트), 클릭하면 관련 프로젝트로 연결

6. 사이드 프로젝트 라이브 데모
AI 앱, Next.js 웹유틸을 실제 동작하는 데모로 보여주기
